# Web Scraper - Data Extraction Pipeline

A lightweight, robust Python web scraper designed to extract product details, pricing, and availability across multiple pages. Built using `BeautifulSoup` and `requests`.

## Features
- **Pagination Handling:** Automatically iterates through multiple pages.
- **Data Export:** Cleans and exports structured data directly into a CSV format.
- **Polite Scraping:** Implements rate-limiting (`time.sleep`) to respect target server bandwidth.

## Tech Stack
- Python 3
- BeautifulSoup4
- Requests
